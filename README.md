# Next.js Boilerplate

## 이 파일을 받아서 Next.js 프로젝트를 시작하세요.😄

### 설치하기

```bash
$ yarn # yarn install
```

### 개발 서버 실행하기

```bash
$ yarn dev
```

### 테스트 실행하기

```bash
$ yarn test
```

### Watch 모드로 테스트 실행하기

```bash
yarn test:watch
```

### 빌드하기

```bash
$ yarn build
```

## ⚠ 주의
이 Boilerplate는 Pre Commit을 사용하고 있습니다. 그래서 Lint나 Test가 통과되지 않으면, 커밋이 불가능합니다.
